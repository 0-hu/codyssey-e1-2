# 🎯 애니메이션 퀴즈 게임

일본 애니메이션을 주제로 한 터미널 기반 퀴즈 게임입니다.

## 📌 프로젝트 개요

Python으로 구현한 콘솔 퀴즈 게임으로, 나루토·원피스·블리치·주술회전 등 유명 일본 애니메이션에 대한 퀴즈를 풀 수 있습니다.

## 🎨 퀴즈 주제 선정 이유

일본 애니메이션은 전 세계적으로 사랑받는 콘텐츠이며, 다양한 작품에 대한 지식을 퀴즈 형태로 즐길 수 있도록 구성했습니다. 나루토, 원피스, 블리치, 주술회전, 드래곤볼, 진격의 거인, 귀멸의 칼날 등 대표적인 작품들을 선정했습니다.

## 🚀 실행 방법

```bash
python main.py
```

- Python 3.10 이상 필요
- 외부 라이브러리 불필요 (표준 라이브러리만 사용)

## 📋 기능 목록

| 기능 | 설명 |
|---|---|
| 퀴즈 풀기 | 랜덤 순서로 퀴즈를 풀고 점수를 확인 |
| 퀴즈 추가 | 새로운 퀴즈를 직접 등록 |
| 퀴즈 목록 | 등록된 퀴즈 전체 목록 확인 |
| 퀴즈 삭제 | 등록된 퀴즈 삭제 |
| 점수 확인 | 최고 점수 및 플레이 히스토리 확인 |
| 랜덤 출제 | 문제 순서를 매번 랜덤하게 섞어 출제 |
| 문제 수 선택 | 원하는 문제 수만큼 선택하여 풀기 |
| 힌트 기능 | 문제 풀이 중 힌트 확인 가능 (점수 차감) |

## 💾 데이터 파일 설명

### `state.json`

- **경로**: 프로젝트 루트 디렉터리
- **역할**: 퀴즈 데이터, 최고 점수, 게임 히스토리를 저장
- **인코딩**: UTF-8
- **스키마**:

```json
{
    "quizzes": [
        {
            "question": "문제 텍스트",
            "choices": ["선택지1", "선택지2", "선택지3", "선택지4"],
            "answer": 1,
            "hint": "힌트 텍스트"
        }
    ],
    "best_score": 0,
    "score_history": [
        {
            "date": "2026-04-08 01:30:00",
            "total_questions": 5,
            "correct_answers": 4,
            "score": 80,
            "hints_used": 1
        }
    ]
}
```

- 파일이 없으면 기본 퀴즈 데이터로 자동 생성됩니다.
- 파일이 손상된 경우 안내 메시지 출력 후 기본 데이터로 복구됩니다.
